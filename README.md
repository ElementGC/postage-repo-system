# 邮政货件仓储系统

## 1. 项目设计

### 1.1 设计要求

设计一套仓储管理系统，实现邮政货件的入库、出库、盘点、预警管理、报表管理等功能。

要求：

•1）实现管理系统信息关系的系统化、规范化和自动化；

•2）管理系统可设置预警规则并自动报警；

•3）设计仓储需求历史数据，并根据历史数据开发仓储需求预测程序，实现未来需求预测。
